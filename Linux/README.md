0. Copy `spotify_control` to /opt/
1. `sudo chmod +x spotify_control`
2. Create custom shortcuts for:

    - /opt/spotify_control previous 	--> Super + ,
    - /opt/spotify_control next			--> Super + .
    - /opt/spotify_control playpause	--> Super + /
